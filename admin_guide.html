Sports Orthopedics Admin Panel Guide

1. Introduction

Welcome to the Sports Orthopedics Admin Panel! This system helps you manage the clinic's daily operations, including patient appointments, doctor schedules, and content management.

This guide explains how to perform common tasks. The admin panel has a navigation menu on the left side with different sections:

  • Dashboard - Overview of clinic activity
  • Doctors - Manage doctor profiles
  • Schedule - Manage doctor availability
  • Appointments - View and manage patient appointments
  • Analytics - View reports and statistics
  • Settings - Configure system settings
  • Content - Manage website content
  • Gallery - Manage image gallery

2. Getting Started

How to Log In

Access the secure admin area.

  1. Open your web browser and go to the admin login page (your supervisor will provide the link, it usually ends with /admin/login).
  2. Enter your email address: admin@sportsorthopedics.in
  3. Enter your password: admin@123
  4. Click the 'Login' button.
Tip: Keep your password confidential. If you forget your password, contact your supervisor.

How to Log Out

Securely exit the admin panel when you are finished.

  1. Look for the 'Logout' button at the bottom of the left sidebar.
  2. Click the 'Logout' button.
Important: Always log out when you finish your work or step away from the computer, especially on shared computers.

Understanding the Dashboard

Get a quick overview of important information after logging in.

After logging in, you'll see the Dashboard with:

  • Stats Cards - Shows today's appointments, upcoming appointments, completed appointments, and cancelled appointments.
  • Quick Actions - Provides shortcuts to common tasks like managing schedules, doctors, appointments, and analytics.
  • Recent Activity - Lists the most recent appointment bookings.

Use the navigation menu on the left to move to specific sections for more details.

3. Managing Appointments

Viewing the Appointment Calendar

See scheduled appointments in a calendar view.

  1. Click on 'Appointments' in the main navigation menu.
  2. By default, you'll see a calendar view showing all appointments for the current month.
  3. Click on a specific day to view all appointments for that day.
  4. Use the view toggle in the upper right (calendar/list icons) to switch between calendar and list views.

Viewing Appointments in List View

See appointments in a filterable list format.

  1. Go to the 'Appointments' section.
  2. Click the list view icon in the upper right corner (looks like three lines).
  3. You'll see a table of appointments that you can sort and filter.
  4. Use the pagination controls at the bottom to navigate through pages of appointments.

Booking a New Appointment

Schedule an appointment for a patient, for example, during a phone call.

  1. Go to the 'Appointments' section.
  2. Click the 'New Appointment' button at the top of the page.
  3. In the modal that appears, fill in the following information:
    • Patient Name
    • Email
    • Phone Number
    • Select a Doctor from the dropdown
    • Select a Date using the calendar
    • Select an available Time Slot
  4. Click 'Save' to create the appointment.

Flow: Booking a New Appointment

1. Start → Go to 'Appointments' section.
2. → Click 'New Appointment' button.
3. → Enter Patient Details (Name, Email, Phone).
4. → Select Doctor.
5. → Select Date.
6. → Select Time Slot.
7. → Click 'Save'.
8. → End (Appointment is created).
Tip: The system will only show available time slots based on the doctor's schedule. Gray or disabled time slots are unavailable.

Updating an Appointment's Status

Mark an appointment as confirmed, completed, cancelled, or no-show.

  1. Find the appointment you want to update (in either calendar or list view).
  2. Click on the appointment to view its details.
  3. In the appointment details view, look for the status dropdown.
  4. Click on the status and select the new status from the options:
    • Scheduled - Initial booking state
    • Confirmed - Appointment has been confirmed
    • Completed - Patient visit completed
    • Cancelled - Appointment was cancelled
    • No Show - Patient didn't attend
  5. The status will update automatically.
Important: Keeping appointment statuses up-to-date is essential for accurate reporting and tracking.

Editing an Existing Appointment

Change the details of an already scheduled appointment.

  1. Find the appointment you want to edit.
  2. Click on the appointment to open its details.
  3. Click the 'Edit' button.
  4. Make the necessary changes to the patient information, doctor, date, or time.
  5. Click 'Save' to update the appointment.

Booking an Appointment from the Calendar View

Quickly add an appointment for a specific date directly from the calendar.

  1. Go to the 'Appointments' section and make sure you're in calendar view (click the calendar icon in the top right if needed).
  2. Click on the specific date you want to book an appointment for.
  3. A side panel will open showing all appointments for that day.
  4. Look for the plus icon (+) in the side panel and click it.
  5. The appointment creation form will appear with the selected date already filled in.
  6. Complete the remaining fields (patient details, doctor, time slot).
  7. Click 'Save' to create the appointment.
Tip: This method is especially useful when a patient calls asking for availability on a specific date, as you can immediately see existing appointments and available slots for that day.

4. Managing Doctors

Viewing All Doctors

See a list of all doctors in the clinic.

  1. Click on 'Doctors' in the main navigation menu.
  2. You'll see cards for each doctor showing their name, specialty, and fee.
  3. Use the search bar at the top to filter doctors by name or specialty.
  4. Use the filter button to filter by specific specialties.

Adding a New Doctor

Add a new doctor to the system.

  1. Go to the 'Doctors' section.
  2. Click the 'Add Doctor' button at the top right.
  3. Fill in the doctor's information in the modal that appears:
    • Name
    • Specialty
    • Consultation Fee
    • Upload a photo (optional)
  4. Click 'Save' to add the doctor.
Note: After adding a doctor, you'll need to set up their schedule separately in the Schedule section.

Editing a Doctor's Information

Update a doctor's details.

  1. Go to the 'Doctors' section.
  2. Find the doctor you want to edit.
  3. Click the menu (three dots) button on their card and select 'Edit'.
  4. Update the necessary information.
  5. Click 'Save' to update the doctor's profile.

Managing a Doctor's Schedule

View or edit when a doctor is available for appointments.

  1. Go to the 'Doctors' section.
  2. Find the doctor whose schedule you want to manage.
  3. Click the menu (three dots) button on their card and select 'Schedule'.
  4. You'll be taken to a page showing the doctor's weekly schedule.
  5. To edit their schedule, click the 'Edit Schedule' button.
  6. For each day of the week, you can set:
    • Working hours (start time and end time)
    • Slot duration (how long each appointment lasts)
    • Buffer time (time between appointments)
    • Break times (lunch or other breaks)
    • Whether the day is active (doctor works that day)
  7. Click 'Save' to update the schedule.

Flow: Setting Up a Doctor's Schedule

1. Start → Go to 'Doctors' section.
2. → Find doctor and click menu → Select 'Schedule'.
3. → Click 'Edit Schedule' button.
4. → For each day of the week:
    a. Set working hours
    b. Set appointment duration
    c. Set buffer time
    d. Set break times
    e. Toggle day active/inactive
5. → Click 'Save'.
6. → End (Schedule is updated).

5. Managing Special Dates

Blocking Off Time for Holidays or Special Events

Mark dates when the clinic is closed or has special hours.

  1. Click on 'Special Dates' in the main navigation menu (or navigate there from Schedule page).
  2. Click the 'Add Special Date' button.
  3. Fill in the details:
    • Title (e.g., "Christmas Holiday", "Staff Training")
    • Start Date and End Date (for multi-day events)
    • All Day - toggle if it's a full-day event
    • If not all day, set Start Time and End Time
    • Description (optional)
  4. Click 'Save' to create the special date.
Important: Special dates block appointments for ALL doctors. For doctor-specific time off, use the doctor's individual special dates.

Managing Doctor-Specific Special Dates

Block time for individual doctors (vacations, conferences, etc.).

  1. Go to the 'Doctors' section.
  2. Find the doctor and click the menu (three dots).
  3. Select 'Special Dates' from the menu.
  4. Click 'Add Special Date' button.
  5. Fill in the details (similar to regular special dates).
  6. Click 'Save' to create the doctor-specific special date.
Tip: Use this feature when a doctor has a day off but the clinic remains open with other doctors available.
